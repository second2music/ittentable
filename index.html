WHAT'S "ITTEN TABLE"？？

ITTEN TABLEは日本の陶芸家とシリコンバレーの消費者をつなぐプラットフォームを目指しています。
1万2000年もの歴史の中、日本では高品質でオリジナリティに富んだ焼き物が作られてきました。
ITTEN TABLEではその焼き物"一点もの"をシリコンバレーのお客様へお届けします。




日本の陶芸家 ✓ 1万2000年にわたる焼き物の歴史
✓ 高い品質・オリジナリティ
✓ 販路開拓に苦労
✓ 労力に見合わない価格






シリコンバレーの消費者 ✓ 日本食・日本文化への高い興味
✓ 本物志向/丁寧な暮らしへの興味
✓ ホームパーティー文化
✓ 高い収入・購買力

SERVICES

本サービスでは、アプリケーションによるテーブルコーディネーターAIがお客様の食卓の好みを自動で認識します。そしてお客様にITTEN TABLEの豊富な一流陶芸家のネットワーク駆使し、オーダーメイドの焼き物”１点もの”を発注していただきます。そして納品までの工程を”見える化”できるのが本サービスの特徴であります。

特長①
テーブルコーディネーターAI による食卓の好みの診断・マッチング
